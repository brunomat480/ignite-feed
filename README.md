##

#### web

Clone o repositório:
```bash
git clone https://github.com/brunomat480/ignite-feed.git
```

Baixe as dependencias:
```bash
yarn
```
ou
```bash
npm i
```
##

#### web
inicie a aplicação:
```bash
yarn dev
```
ou 
```bash
npm run dev
```
